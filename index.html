Social betting app

November 2019

Betyourmate is an app for solving "Gentlemens' agreements involving money". This mobile-first website was created as a group project using React, Redux, SCSS and a bit of Bootstrap for the front end side. For the backend we used Ruby on Rails. We implemented JWT user authentication. Both sides are hosted on Heroku.

We used Github to manage the project between us. Branching was a lifesaver for trying out new features.
