Markdown Web Editor

December 2019

NotAble is an app which enables user to create, save and edit Markdown documents and see the renders in real time. I created it using React and Firebase. I used SASS for styling. To authenticate users I used methods provided by Firebase. I also implemented their NoSQL Database to store user information and notes.

As a personal project I recreated the same front-end functionality with Svelte

Social betting app

November 2019

Betyourmate is an app for resolving "Gentlemens' agreements involving money". This mobile-first website was created as a group project using React, Redux, SCSS and a bit of Bootstrap for the front end. For the backend we used Ruby on Rails, with Active Storage for storing profile images. We implemented JWT user authentication. Both sides are hosted on Heroku.

The future of the project is on an app and that's what we're working right now. We will be using React Native to build out an app for both systems and AWS/Firebase for the backend

We used Github to manage the project between us. Branching was a lifesaver for trying out new features.

Our code is private, as we are planning to continue working on this app, but here is a Github repo with public Readme which is going into detail of how the app works.
